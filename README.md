Консольна програма на C++ для роботи з таблицями замірів танків судна.
Курсова робота, 1 курс коледжу.

Програма дозволяє визначити об'єм рідини в танку судна за значенням заміру (висота стовпа рідини в сантиметрах).
Дані засновані на офіційному документі.

## Функціонал

-  Перегляд списку всіх танків судна
-  Введення значення заміру в сантиметрах
-  Розрахунок об'єму рідини (м³) з лінійною інтерполяцією
-  Розрахунок маси рідини (т)
-  Визначення координат центру тяжкості (LCG, TCG, VCG)
-  Перегляд повної таблиці замірів обраного танку
-  Перевірка коректності введених даних
